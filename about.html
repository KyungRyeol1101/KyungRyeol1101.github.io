--- layout: null section-type: about title: About --- ## About Github's Octocat was designed by Simon Oxley, alongside the white bird Twitter used (before they received a proper logo) as part of a usual routine of cranking out images for iStock. GitHub saw it, and wanted it, presumably under the notion that it can represent how complex code combines to create peculiar things, much like the octopuss... except the CEO of GitHub called it an octocat, and it has been the octocat since [via]. 다양한 파트의 팀원들과 협업을 즐기고 피드백을 적극적으로 수용하고 개선하는데 즐거움을 느끼고 있습니다. 코드의 양을 줄임 수정하기 쉬운 코드 이해하기 쉬운 코드 재활용하기 좋은 코드
